A commercial license is required if:
- You are building proprietary software that incorporates `space-ml-sim`
- You are deploying a service that uses `space-ml-sim` and do not wish to release your source code
- Your organization's legal policy prohibits AGPL dependencies

## FAQ

**Q: Can I evaluate the software before purchasing a commercial license?**
A: Yes. The AGPL allows full use and evaluation. You only need a commercial license when you deploy or distribute proprietary software that includes `space-ml-sim`.

**Q: Do I need a commercial license if I just run the examples locally?**
A: No. Running the software for personal use, research, or evaluation does not trigger AGPL obligations.

**Q: What if I use space-ml-sim outputs (data, plots) but not the code itself?**
A: Outputs generated by running unmodified `space-ml-sim` are not covered by the AGPL. You own your simulation results.
